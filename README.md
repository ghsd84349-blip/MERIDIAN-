# MERIDIAN — Премиальное интерактивное веб-приложение недвижимости

Интерактивный SPA-портал (Single Page Application) для презентации и бронирования элитных резиденций в Москве, Санкт-Петербурге и Нью-Йорке.

## 🌟 Особенности проекта:
* **Архитектурный минимализм:** Дизайн в стиле Luxury Real Estate, разработанный с упором на типографику и геометрию пространства.
* **Динамическая фильтрация:** Интерактивное переключение локаций и типов сделок (покупка/аренда) на чистом JavaScript без перезагрузки страниц.
* **Многостраничная структура слайдов:** Плавная смена экранов при просмотре объектов и планировок.
* **Компонентная архитектура:** Проект разделен на независимые модули (HTML5, CSS3, чистый JS), что соответствует стандартам коммерческой разработки.

## 🛠 Технологический стек:
* HTML5 (Семантическая разметка)
* CSS3 (Flexbox, Grid, Custom Properties, CSS-анимации)
* JavaScript (Vanilla JS, Асинхронное управление DOM, Event Listeners)
